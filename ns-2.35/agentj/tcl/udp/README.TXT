In the directory there are the following demos:

1) clientserver.tcl - basic udp connection
2) lotsofnodes.tcl - creates and initialises multiple udp clients (500)
3) simpleGroup.tcl - creates a multicast group and uses this group to send messages between 2 particpants.
4) clientserver.tcl - creates a group and initiates a timer to send messages multiple times to that group
